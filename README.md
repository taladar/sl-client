# sl-client

[![GitHub Actions Workflow Status](https://img.shields.io/github/actions/workflow/status/taladar/sl-client/github-release.yaml)](https://github.com/taladar/sl-client/actions/workflows/github-release.yaml)
Rust libraries to connect to Second Life or OpenSim based grids

## Documentation

A high-level tour of the Second Life / OpenSim protocol — and a map of which
crate types implement each part of it — lives in [`book/`](book/), an
[mdBook](https://rust-lang.github.io/mdBook/). Build it locally with:

```sh
cd book && mdbook serve --open
```
