The flycam control floater: enter / leave flycam, speed and flycam-behaviour
controls (Firestorm's Flycam). This floater controls the camera **mode**
([[viewer-camera-flycam]]); it is distinct from
[[viewer-input-spacenav-settings-ui]] (`llfloaterjoystick`), which configures
the 6-DOF input **device** axes.

Reference (Firestorm, read-only): the Flycam controls in `llfloatercamera` /
`llviewerjoystick`.
